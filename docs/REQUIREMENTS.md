# Requirements Index

## Core User Flow

1. User opens the app
2. User is shown a short Portuguese passage containing a target word from their word bank
3. User reads the passage
4. User can:
   - Tap any word to see its translation
   - Mark the target word as learned
   - Recycle the word for another example
   - Reveal the full translation of the passage
