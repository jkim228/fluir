# Tap-to-Translate

## User Story

As a learner, I want to tap any word in a passage to quickly understand its meaning without leaving the reading experience.

---

## Requirements

### Tapping
- [ ] Every word in a generated passage is tappable
- [ ] Tapping a word shows a popover anchored near the tapped word

### Popover Content
- [ ] Popover displays the tapped word in Portuguese
- [ ] Popover displays the English translation of the tapped word

### Dismissal
- [ ] User can dismiss the popover by tapping outside of it
- [ ] User can dismiss the popover by tapping the same word again
